## What is Docker?

+++

Docker is a tool designed to make it easier to create, deploy, and run applications by using containers.

 - Configuration Simplification
 - Pipeline Management (CI/CD)
 - App Isolation
 - Rapid Deployment
 - Closer to Production Environment
